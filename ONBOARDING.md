> Быстрый вход в проект для новой сессии. Актуально на 2026-07-04, версия сайта **2.0**.

## Что это
Интерактивная кадастровая карта **с. Бухта** (Генический район, Херсонская область) — коса Бирючий/Федотова.
Одностраничник на Leaflet, без бэкенда. Данные вручную обогащаются из уведомлений о присвоении РФ-адресов.

⚠️ **с. Бухта = Генический р-н, Херсонская обл.** НЕ Кирилловка, НЕ Запорожье (старые укр. привязки убраны — не возвращать).

## Поля участка (councils JSON)
`c` кадастр укр · `a` площадь га · `o` владелец · `ot` форма собств. · `zn` массив ·
`fias` адрес РФ («улица X, земельный участок N») · `building` номер здания · `bcadrf` кадастр здания РФ ·
`cadrf` кадастр участка РФ · `egrn` («да»=оформлен) · `post` № постановления · `ct` ВРИ

## Карта / UI
- Старт: центр = участок `11:003:0238`, zoom **16.3**, minZoom **15**, maxBounds вокруг 0238 (за края не выйти)
- Цвета: 🔵 голубой = ЕГРН · 🩷 розовый = державна власність · 🟢 салатовый = прочие; обводки жёлтые; крупные (>0.2 га) — слабая заливка (вложенные видны)
- 🚩 красный флажок по центру участка = есть адрес РФ (fias)
- Попап: кадастр+копи, площадь, массив, владелец, 🇷🇺 РФ-кадастр+копи, ✓ЕГРН, адрес РФ, [рамка] здание+кадастр здания, постановление
- Правый контрол: слои (Схема тёмная/светлая/Спутник) + фильтр ЕГРН (Все / Только с ЕГРН / Без ЕГРН / 🚩 С адресами ХО)
- Поиск: кадастр (укр/РФ, полный/фрагмент 4+ цифр), ФИО, адрес токенами («набережная 76»)
- Клик = самый мелкий участок под точкой (геометрический hit-test — вложенные кликабельны)
- «Последние просмотренные» слева (10, localStorage)
- Мобильная версия: бургер ☰ (панель) + ⚙ (слои/фильтр), zoom под бургером, тап по затемнению закрывает
- Тех-инфо внизу справа: `версия · зум` (BUILD_VERSION в index.html — бампить при каждом деплое)

## Рабочий процесс
1. **Пользователь присылает уведомления** (сканы/фото/xlsx/pdf) — извлечь кадастр (укр `2320355400:…` или РФ `96:01:…`), адрес (улица + ЗУ N), здание (если «дом/здание N»), № постановления.
   - PDF-сканы: `pymupdf` → рендер в PNG → читать визуально (poppler нет). Повёрнутые страницы — доворачивать матрицей.
2. **Матч с базой** по укр-кадастру (`p.c`) или РФ (`p.cadrf`). Суффикс бывает в двух кварталах (12:001 и 11:003) — разрешать по владельцу/адресу.
3. **Проверять крупным планом** — беглое чтение сканов даёт ошибки (путал 26↔274, 169/2↔169/3).
4. **Владельца менять** только по явной просьбе. ФИО-сокращения — как просит пользователь.
5. После правки: `commit + push`, **бампить BUILD_VERSION**, дать ссылку `?v=<sha>`.
6. Проверять на проде через `curl …?cb=$(date)` — у пользователя часто кэш браузера.
