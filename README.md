# buhta

Интерактивная кадастровая карта с. Бухта (Генический район, Херсонская область, коса Бирючий/Федотова).

Одностраничник на Leaflet, без бэкенда: 524 участка, привязка к российским кадастрам и адресам ФИАС, статусы регистрации в Росреестре, реестр КУВД. Данные обновляются вручную из уведомлений и выписок.

Сайт: https://x270880x.github.io/buhta/

Контекст для работы над проектом — в `CLAUDE.md`, `ONBOARDING.md` и `PROJECT_LOG.md`.
