# CLAUDE.md — Бухта, кадастровая карта

Контекст проекта. Подробности — в `ONBOARDING.md` и `PROJECT_LOG.md` (читать при старте сессии).

## Суть
Интерактивная кадастровая карта **с. Бухта** — Leaflet-одностраничник, без бэкенда.
⚠️ **с. Бухта = Генический район, Херсонская область.** НЕ Кирилловка, НЕ Запорожье — не возвращать старые укр. привязки.

## Структура
- `index.html` — вся карта (Leaflet 1.9.4 + leaflet-rotate, bearing 60°)
- `councils/2320355400.json` — метаданные участков
- `parcels/2320355400.geojson` — геометрия (522 полигона)
- `ONBOARDING.md` / `PROJECT_LOG.md` — полный контекст и история

## Поля участка (councils JSON)
`c` кадастр укр · `a` га · `o` владелец · `ot` форма собств. · `zn` массив ·
`fias` адрес РФ · `building` здание · `bcadrf` кадастр здания РФ · `cadrf` кадастр участка РФ ·
`egrn` («да») · `post` постановление · `ct` ВРИ · `rr` («да» = (пере)регистрация в Росреестре по Херсонской обл. → зелёный флажок) ·
`zaddr` старый запорожский адрес (из докс-таблицы ФИАС; НЕ выводить в UI — хранится скрыто по просьбе пользователя) ·
`brr` («да» = сам ДОМ перерегистрирован в Росреестре — считается в статистике домов; rr участка на дом не распространяется) ·
`mfc`/`bmfc` («да» = участок/дом подан на регистрацию в МФЦ, ждёт; после регистрации ставить rr/brr и снимать mfc/bmfc).
Топ-левел `unbound` в JSON — дома без привязки к укр. участку (Набережная 46 А/Б/В); видны только в статистике, на карту попадут после привязки.

## Правила работы
1. Обработка уведомлений: извлечь кадастр (укр `2320355400:…` / РФ `96:01:…`) + адрес (улица+ЗУ N) + здание + № постановления. Матч по `p.c` или `p.cadrf`. Суффикс бывает в кв. 12:001 и 11:003 — разрешать по владельцу/адресу.
2. PDF-сканы: `pymupdf` → PNG → читать визуально. **Проверять крупным планом** — беглое чтение даёт ошибки.
3. Владельца менять только по явной просьбе.
4. После правки: `commit + push`, **бампить BUILD_VERSION в index.html**, дать ссылку `?v=<sha>`.
5. Коммиты/код/пути — на английском; общение — по-русски.
6. Жукова (если появится) → фактическая хозяйка Моршнева Н.В.
